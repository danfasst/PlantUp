# PlantUp

## Descrição

**PlantUp** é um projeto em desenvolvimento com o objetivo de auxiliar os usuários a cuidar de suas plantas e jardins. O aplicativo oferece lembretes para regar, dicas de cuidados personalizados e tutoriais de jardinagem, tornando o processo de cuidar das plantas mais simples e eficaz.

## Metas de Desenvolvimento

1. **Interface do Usuário (UI/UX)**  
   - [ ] Criar uma interface de usuário simples, intuitiva e amigável para facilitar a navegação.
   
2. **Funcionalidades de Lembrete**  
   - [ ] Implementar sistema de lembretes automáticos para regar plantas.
   - [ ] Incluir notificações para outros cuidados, como poda, adubagem e troca de vasos.

3. **Dicas Personalizadas de Cuidados**  
   - [ ] Desenvolver uma base de dados com dicas específicas para diferentes tipos de plantas.
   - [ ] Integrar essas dicas ao app, oferecendo recomendações com base nas plantas cadastradas pelo usuário.

4. **Tutoriais de Jardinagem**  
   - [ ] Adicionar tutoriais detalhados para iniciantes em jardinagem.
